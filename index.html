Padding Margin Border
Padding refers to the space between the element content and the element border. The Margin property is used to define the space around elements. This is used to set the Border on an element
Padding gives space to the top, left, right and the bottom between our Text and the Border. It clears an area around the element. The CSS border properties (border-style, border-color, border-width and border-radius) are used to specify the style, color and size of the border of an element
